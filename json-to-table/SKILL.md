# JSON to Table 转换助手

## 激活场景
当用户提供了一串 JSON 代码，并要求“整理一下”、“看清楚点”或“做成表格”时，激活此技能。

## 执行步骤
1. **字段过滤**：忽略所有以 `_` 开头的元数据字段（如 `_version`, `_id`）。
2. **表头生成**：将剩余字段的首字母大写作为表头。
3. **数据填充**：将数值转换为字符串填入。
4. **特殊处理**：如果字段名包含 `status`，根据值添加 Emoji：
    - `success` -> ✅
    - `pending` -> ⏳
    - `error` -> ❌

## 限制
如果 JSON 是嵌套的深层结构，仅提取第一层级的数据进行展示。
